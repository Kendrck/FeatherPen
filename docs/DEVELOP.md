# DEVELOP.md FeatherPen开发调试指南 V1.0.0
## 1. 本地环境初始化
运行根目录 init_env.py，自动创建虚拟环境、安装依赖、初始化空数据目录。

## 2. 测试账号调试流程
1. 打开登录窗口，使用下方快捷按钮自动填充UID与密码
2. Lv1~Lv8账号：正常扣除积分，梯度开放功能上限
3. Lv9账号：自动展示积分豁免开关面板，可切换免扣费状态

## 3. 打包发布规范
1. dist目录加入.gitignore，不提交代码仓库
2. 打包完成后上传产物至GitHub Releases分发
3. 使用Git LFS管理超大资源文件（图片、字体、安装包）

## 4. CI/CD自动化
.github/workflows配置自动单元测试、跨平台打包脚本。
