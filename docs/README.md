# FeatherPen 羽笔 - 官方开发文档 (V1.0.0)

FeatherPen 是一款基于 Python 的跨平台 AI 长篇小说生成工具，采用“世界树”标准仓库架构。本项目优先本地离线运行，同时支持云端 API 协同。

## 🚀 核心特性
- **三级创作体系**：严格遵循 卷(Volume) -> 章(Chapter) -> 节(Section) 结构。
- **双模登录**：支持本地离线游客 (Lv0) 与 云端白名单账号登录。
- **特权体系**：内置 10 组固定测试账号，Lv9 账号支持“积分豁免”开关。
- **全平台兼容**：Windows/Linux/macOS 统一 UI 交互与逻辑。

## 🛠️ 快速启动
1. 初始化环境：`python init_env.py`
2. 启动应用：`python main.py`
