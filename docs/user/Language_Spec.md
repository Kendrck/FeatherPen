# FeatherPen 多语言支持规范 V1.0
## 一、概述
本文档定义软件多语言切换、语言包扩展、跨平台兼容、异常降级统一标准。

## 二、默认支持语种
软件默认语言：简体中文(zh-CN)
内置支持4种官方语种：
- zh-CN 简体中文
- en-US 美式英语
- fr-FR 法语
- es-ES 西班牙语

## 三、语言包格式
支持两种标准语言扩展格式：
- *.json 标准翻译文本包
- *.langpack 整合式语言资源包

## 四、部署方式
1. 将语言包放入 /lang/对应语种文件夹
2. 无需编译、无需配置
3. 重启软件即可生效

## 五、目录标准结构
```
lang/
├── zh-CN/strings.json
├── en-US/strings.json
├── fr-FR/strings.json
└── es-ES/strings.json
```

## 六、跨平台兼容规则
1. 全平台统一 UTF-8 编码
2. 桌面、Web、移动端共用同一套语言包
3. 小程序等受限环境自动降级内置中文

## 七、容错降级规则
1. 语言包损坏 → 自动回退简体中文
2. 翻译Key缺失 → 显示原始文本
3. 语言切换不影响用户数据、配置、项目文件
