# FeatherPen V1.0.0 API 接口规范文档

**文档版本**：V1.0.0
**最后更新**：2026-07-20
**维护部门**：后端开发组

---

## 一、文档说明

本文档定义 FeatherPen 羽笔 V1.0.0 所有后端 API 接口的完整规范，包括请求格式、响应格式、错误码、认证方式等。所有接口遵循 RESTful 风格，统一使用 JSON 数据格式。

---

## 二、通用规范

### 2.1 接口前缀

所有业务接口固定前缀：`/api/v1/`，禁止自定义前缀。

### 2.2 通信协议

| 类型 | 协议 | 说明 |
|------|------|------|
| 普通请求 | HTTP POST/GET | 同步请求，立即返回结果 |
| 流式生成 | SSE (Server-Sent Events) | 低延迟流式输出，实时返回生成片段 |

### 2.3 数据格式

- 所有请求和响应：**JSON** 格式
- 字符编码：**UTF-8**
- 时间格式：**ISO 8601**（如 `2026-07-20T14:30:00Z`）

### 2.4 错误处理规范

**标准HTTP状态码**：

| 状态码 | 含义 |
|--------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | 未认证 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

**自定义业务错误码**：

| 错误码 | 错误名称 | 说明 |
|--------|----------|------|
| 4001 | Model Not Loaded | 推理引擎未加载模型或模型文件损坏 |
| 4002 | Invalid Context ID | 上下文ID不存在或已过期失效 |
| 4003 | Insufficient Points | 积分余额不足，无法完成操作 |
| 4004 | Daily Limit Exceeded | 当日积分消耗已达上限 |
| 4005 | Permission Denied | 当前会员等级无此操作权限 |
| 5001 | GPU OOM | GPU显存溢出，需降低生成参数或切换量化模型 |
| 5002 | Engine Timeout | 推理引擎响应超时 |
| 5003 | Engine Busy | 生成任务队列已满，请求排队重试 |

**统一异常返回结构**：

```json
{
  "code": 4001,
  "message": "Model Not Loaded",
  "detail": "推理引擎未加载模型或模型文件损坏",
  "timestamp": "2026-07-20T14:30:00Z"
}
